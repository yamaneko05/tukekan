# 1. プロジェクト概要・ユーザー要件

## 1.1 目的

友人間のお金の貸し借りを管理するWebアプリケーション。スマホからの入力のしやすさと、貸借状況の可視化を重視する。

## 1.2 背景

現状はスプレッドシート（CSV）で運用しているが、以下の課題がある：
- スマホからの入力が煩雑
- 誰にいくら貸しているか/借りているかが一目でわかりにくい
- 複数ユーザーでの共有が難しい

---

## 2. ユーザー要件

### 2.1 ユーザーの種類

- **登録ユーザー**: アプリにログインして自分の貸借を管理するユーザー

### 2.2 認証要件

| 項目 | 内容 |
|------|------|
| 認証方式 | 自前実装（bcrypt + JWT） |
| データの所有者 | 各ユーザーが自分のTransactionを所有 |
| データの可視性 | 全ユーザーの全Transactionを閲覧可能（※将来的にグループ単位で制限） |

### 2.3 ユーザーストーリー

1. **自分の貸借を管理したい**
   - 自分が誰にいくら貸しているか/借りているかを記録できる
   - 過去の取引履歴を確認できる

2. **他のメンバーの貸借も確認したい**
   - 他ユーザーの貸借残高一覧を閲覧できる
   - 他ユーザーの取引履歴を閲覧できる

3. **簡単に入力したい**
   - スマホから素早く貸借を登録できる
   - よく使う相手や項目をサジェストしてほしい
