# 2. 機能要件

## 2.1 MVP機能（初期リリース）

### 認証機能

- [ ] ログイン/ログアウト

※ 新規登録はPhase 3（グループ機能）で実装。MVPでは初期ユーザーをseedで作成。

### ホーム画面（タブ切替）

- [ ] **残高タブ**: 総貸借残高 + 相手ごとの残高一覧
- [ ] **履歴タブ**: 自分の全取引を時系列で一覧表示
- [ ] 相手をタップで取引履歴画面へ遷移

### メンバー画面

- [ ] 全ユーザーの一覧表示（総残高付き）
- [ ] タップで他ユーザーのダッシュボード画面へ遷移

### 取引登録（モーダル）

- [ ] 相手の選択（登録済みPartner or 新規作成）
- [ ] 金額入力（プラス=貸し、マイナス=借り/返済）
- [ ] 説明（任意、過去履歴からサジェスト）
- [ ] 日付（デフォルト: 今日）

### 取引履歴画面

- [ ] 特定の相手との取引履歴一覧
- [ ] 現在の残高表示

### 設定画面

- [ ] プロフィール編集（表示名）
- [ ] ログアウト

### ナビゲーション

- [ ] ボトムバー（ホーム、メンバー）
- [ ] FAB（取引登録）
- [ ] ヘッダー（設定アイコン）
